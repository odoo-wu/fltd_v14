Description

Configuration Naming, Sequence, Category and Attribute Features

You can Enable or Disable feature based on your requirements.

(1) Enable naming configuration for set product name length and it's followed by the symbol.
(2) Enable attribute configuration for set product variant name length and it's followed by the symbol.
(3) Enable category configuration for set product category name length and it's followed by the symbol.
(4) Enable sequence for add sequence at the end of the reference code.
